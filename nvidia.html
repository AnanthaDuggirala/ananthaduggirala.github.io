Work involves:
• Initially I was trained on a project called Detection which will be identifying the objects in the images. Such as Cars, Pedestrians, Cycles, Riders, Traffic signals, Other objects.
• My work is to analyze the images that are tagged and send 5% of the images from the project, produce JSON file for each image tagged which has metadata helpful in training the deep learning framework.
• Once the JSON is created I will train the network using deep learning frameworks (Python Scripts). Later once training is finished testing will be performed.
• Initially test is performed on already trained images.
• Later based on training and testing accuracy, New images are used for predicting on which four outcomes are possible.
• Outcomes are True Positive, True Negative, False Positive, False Negative.
• Once the results are out are saved and exported for data visualization, generated on BI tool for further analysis.
